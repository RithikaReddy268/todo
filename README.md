# To-Do List Application

## Overview

This is a simple command-line To-Do List application developed in Python. It allows users to manage daily tasks by adding, viewing, and removing activities from a task list.

## Features

* View all tasks
* Add new tasks
* Remove existing tasks
* Exit the application

## Technologies Used

* Python

## How It Works

When the program starts, a menu is displayed with the following options:

1. Show Tasks
2. Add Task
3. Remove Task
4. Exit

Users can choose an option by entering a number from 1 to 4.

### Show Tasks

Displays all tasks currently stored in the list.

### Add Task

Allows the user to add a new activity to the task list.

### Remove Task

Removes a specified activity from the task list.

### Exit

Closes the application.

## How to Run

1. Make sure Python is installed on your system.
2. Save the program as `todo_list.py`.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the command:

```bash
python todo_list.py
```
